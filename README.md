# ✨ Particle Playground

An interactive, customizable particle system built with HTML, CSS, and JavaScript. Create beautiful visual effects by interacting with particles on the canvas.

## Features

- 🎨 **Interactive Particle System** - Click and drag to create particle effects
- 🌈 **Multiple Color Modes** - Rainbow, Monochrome, Fire, Ocean, and Custom colors
- ⚙️ **Customizable Controls** - Adjust particle count, size, speed, and connection distance
- 🖱️ **Mouse Interaction** - Particles react to your mouse movement
- 📱 **Responsive Design** - Works on desktop and mobile devices

## How to Use

1. Open `index.html` in your web browser
2. Click anywhere on the canvas to add particles
3. Drag your mouse to create particle trails
4. Move your mouse near particles to see them react
5. Use the controls to customize the particle behavior:
   - **Particle Count**: Number of particles on screen (50-500)
   - **Particle Size**: Size of each particle (1-10)
   - **Speed**: Movement speed multiplier (0.5-5)
   - **Connection Distance**: Distance at which particles connect (50-200)
   - **Color Mode**: Choose from different color schemes
   - **Reset Canvas**: Regenerate particles with current settings
   - **Clear Particles**: Remove all particles

## Color Modes

- **Rainbow**: Gradient colors based on horizontal position
- **Monochrome**: Grayscale gradient
- **Fire**: Warm orange/red tones
- **Ocean**: Cool blue/cyan tones
- **Custom**: Choose your own color

## Technologies Used

- HTML5 Canvas API
- CSS3 (Gradients, Animations, Flexbox, Grid)
- Vanilla JavaScript (ES6 Classes, Event Listeners)

## Browser Compatibility

Works best in modern browsers that support:
- HTML5 Canvas
- CSS3 Gradients
- ES6 JavaScript
